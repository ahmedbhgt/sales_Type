## Project Overview
This project Sales_Customer_Type performance, customer behavior, product profitability, and discount impact using Power BI, SQL, and Python.

## Tools Used
- SQL
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Key Insights
- Customers tend to purchase both high and low-priced products.
- Discounts increased quantity sold but negatively impacted profit margins.
- Clothing category performed best during discount campaigns.
- Food category achieved high AOV but lower quantities.
- Returning customers generated stable revenue over time.

## Business Recommendations
- Improve discount strategy to protect profitability.
- Use Upselling and Cross-selling to increase Average Order Value (AOV).
- Create Bundles for low-performing products.
- Encourage Credit Card payments using cashback offers.

## Dashboard Features
- Revenue Analysis
- Profit Analysis
- Customer Segmentation
- Product Performance
- Discount Impact Analysis
- Payment Method Analysis
